# Cool-Bains-manageTerminall
“Cool Bains manageTerminall”technical support
“酷球即刻-管理端”是管理视频发布、已发布视频以及管理“收藏”、“评论”等信息的应用。
主要功能：
1、收藏管理-查看用户收藏视频的统计；
2、视频发布-往用户端发布教学视频；
3、视频管理-查看已发布到用户端的视频，删除无效或者过时视频；
4、评论管理-查看用户评论；
5、消息管理-查看系统消息；
6、管理端账户信息-对管理端账户信息的查阅和修改；
